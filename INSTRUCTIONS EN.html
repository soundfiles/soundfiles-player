SoundFiles.eu Player Pro v5.1

This plugin enables automatic integration of the SoundFiles.eu audio player directly onto your WordPress website.

1. Installation

  • Ensure the plugin folder is named soundfiles-player.
  • Upload the folder to the /wp-content/plugins/ directory.
  • Activate the plugin through the 'Plugins' menu in WordPress.

2. Usage Methods

A. Automatic Link Replacement

Simply paste a direct file link into your post content. The plugin will detect it and convert it into a player:

https://soundfiles.eu/FILE_ID/file

B. Shortcode [soundfile]

For more control, use the shortcode. You can insert it into posts, pages, or widgets.

  • Standard: [soundfile id="FILE_ID"]
  • Custom Color: [soundfile id="ID" color="#ff0000"]
  • Custom Width: [soundfile id="ID" width="500px"]
  • Custom Button Text: [soundfile id="ID" text="DOWNLOAD NOW"]

3. Global Configuration

Navigate to Settings » SoundFiles Player to define default parameters for all players:

  • Color: Changes the color of the Play button, waveform, and download button.
  • Width: Sets the default player size (recommended: 800px or 100%).
  • Button Text: The global label for the download button.
  • Download Toggle: Allows you to show or hide the download button below the player.

4. Troubleshooting

  • Player width is not changing: After saving settings, always refresh your page using Ctrl + F5 (force cache clear).
  • Player is aligned to the left: The plugin uses Flexbox for centering. Ensure your theme does not block !important styles.
  • PHP Error: This plugin requires PHP version 7.4 or higher.
