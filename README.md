== Description ==

SoundFiles.eu Player Pro is a lightweight and powerful plugin that automatically transforms your SoundFiles.eu links into a beautiful, interactive audio player. 

Key features:
* **Automatic Conversion:** Just paste a link like `https://soundfiles.eu` and it turns into a player.
* **Customizable Styles:** Change the player and button colors directly from the admin panel.
* **Responsive Design:** Works perfectly on mobile, tablets, and desktops.
* **Shortcode Support:** Use `[soundfile id="CODE"]` for advanced placements.
* **Custom Download Button:** Control the visibility and text of the download link.
* **Perfect Centering:** Uses modern Flexbox to ensure the player looks great on any theme.

== Installation ==

1. Upload the `soundfiles-player` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your preferences in **Settings > SoundFiles Player**.

== Frequently Asked Questions ==

= How do I embed a player? =
Simply paste the link to your file (e.g., https://soundfiles.eu) on a new line in your post editor.

= Can I change the player width? =
Yes, you can set a global width (like 800px or 100%) in the settings page.

= Does it support shortcodes? =
Yes! You can use `[soundfile id="qPez4jjemOYn"]`. You can also override global settings: `[soundfile id="ID" color="#ff0000" width="500px" text="Get File"]`.
